# Contributing to Language Core Agent

First off, thank you for considering contributing to the Language Core Agent! It's people like you that make learning more accessible for everyone.

## Code of Conduct

By participating in this project, you agree to abide by our Code of Conduct (be kind, respect others, and keep it professional).

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please open an issue and include:
- A clear, descriptive title.
- Steps to reproduce.
- Expected vs. actual behavior.
- Screenshots if applicable.

### Suggesting Enhancements
Have an idea for a new feature? We'd love to hear it! Open an issue with the "enhancement" tag.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. Install dependencies (`npm install` in `web/`, `pip install` in `api/`).
3. Ensure your code follows the established design system (OKLCH, modern typography).
4. Update documentation if necessary.
5. Open a PR with a detailed description of your changes.

## Design Guidelines
- Use **OKLCH** colors for all new UI components.
- Maintain the **Luxury/Professional** aesthetic (minimalist, glassmorphism).
- Ensure all components are responsive and accessible.

## Questions?
Feel free to reach out via GitHub issues. Happy coding!
